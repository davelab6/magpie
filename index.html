Magpie: A Github Project Management Extension for Google Apps

Magpie allows you to track your github projects using a Google Spreadsheet -- all you have to do is link a spreadsheet to a specific milestone on your Github project and it will be updated hourly.

Get from Github »

Magpie is built on three layers. If you're only interested in using Magpie (and not development), skip directly to this section.

GASP!

The lowest layer is an extension that is to Google App Script what jQuery is to Javascript. Tongue in cheek, it's called GASP - Google App Script Plugin.

Installation

  1. Create a new Google Spreadsheet.
  2. From the menu, select Tools > Script Editor...
  3. Create a new script project for Spreadsheet.
  4. In the script editor window menu, select Resources > Libraries...
  5. Find Magpie using the following project key: 
    MkAKgftxvg2TEFXUGomqj19ZrzsJqypJa
  6. Select the latest version and provide an identifier (I've used the identifer include) in my examples.
  7. Switch on debug if you want to develop on top of Magpie.
  8. Save
